## Content Templates

### Blog post template
1. **Title:** Problem-oriented or tutorial-focused
2. **Intro (1 para):** Why this matters, what the reader gets
3. **Body:** Code snippets (max 3 screens), diagrams (ASCII or generated), screenshots
4. **Outro:** Related reads, try it yourself link, CTA (GitHub star / join Discord)

### Twitter thread template
1. Hook (1 tweet) — provocative question or bold statement
2. Problem (1–2 tweets) — what's broken today
3. Solution (2–3 tweets) — how Agent Messenger fixes it
4. Proof (1 tweet) — screenshot, demo, stat
5. CTA (1 tweet) — link + ask (star, try, join)

### Newsletter template
- **Subject:** Monthly roundup: [theme]
- **Body:** 3–5 highlights (new project, blog, community win, coming soon)
- **Footer:** Links to Discord, GitHub, docs, @agent-messenger
